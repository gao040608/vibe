# 可用工具

## read_file — 读取文件
```tool_call
{"name": "read_file", "arguments": {"file_path": "路径"}}
```

## create_file — 创建文件
```tool_call
{"name": "create_file", "arguments": {"file_path": "路径", "content": "内容"}}
```

调用工具时只输出 tool_call 块，不要任何自然语言。

# 技能文件位置
所有技能模板存储在 backend/skills/ 目录下，文件名格式为 技能名.md
例如：backend/skills/landing-page.md
